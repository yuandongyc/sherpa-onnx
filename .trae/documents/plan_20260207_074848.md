## 创建 audio-transcriber-tiny-v4 项目

### 任务概述
基于 `audio-transcriber-tiny` 项目创建 `audio-transcriber-tiny-v4` 项目，使用 Silero VAD v4 模型进行语音活动检测。

### 实施步骤

#### 1. 创建项目目录结构
在 `audio-transcriber-all` 文件夹下创建 `audio-transcriber-tiny-v4` 目录：
- `src/` - 存放源代码
- `files/` - 存放转录结果

#### 2. 复制项目文件
从 `audio-transcriber-tiny` 复制所有文件到 `audio-transcriber-tiny-v4`：
- `src/index.ts`
- `src/transcriber.ts`
- `src/sherpa-onnx-node.d.ts`
- `package.json`
- `tsconfig.json`

#### 3. 修改 package.json
更新 `audio-transcriber-tiny-v4/package.json`：
- name: `audio-transcriber-tiny-v4`
- description: `Audio transcription module using VAD + Whisper (Tiny model + VAD v4)`

#### 4. 修改 test.js
更新 `audio-transcriber-tiny-v4/test.js` 中的 VAD 模型路径：
- 从 `silero_vad.onnx` 改为 `silero_vad_v4.onnx`
- 其他配置保持不变（Tiny Whisper 模型）

#### 5. 构建项目
运行 `npm run build` 编译 TypeScript 代码

#### 6. 运行测试
执行 `node test.js` 对比 v4 和 v3 版本的效果

### 文件结构
```
audio-transcriber-all/
└── audio-transcriber-tiny-v4/
    ├── src/
    │   ├── index.ts
    │   ├── transcriber.ts
    │   └── sherpa-onnx-node.d.ts
    ├── files/           (用于存放转录结果)
    ├── package.json
    ├── tsconfig.json
    └── test.js
```

### 预期改进
使用 Silero VAD v4 模型后，预期会有以下改进：
- 更高的语音检测准确率（99.2% vs 92.3%）
- 更少的误检和漏检
- 更好的噪声环境鲁棒性
- 更精确的语音边界检测
