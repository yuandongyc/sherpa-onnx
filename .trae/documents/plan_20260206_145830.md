# 创建音频识别功能模块

## 新增文件夹和文件

### 1. 创建 audio-transcriber 模块
创建 `audio-transcriber/` 文件夹，包含：
- `src/` - TypeScript 源代码目录
  - `transcriber.ts` - 识别器主类
  - `index.ts` - 导出文件
- `dist/` - 编译输出目录（自动生成）
- `package.json` - 模块配置
- `tsconfig.json` - TypeScript 配置
- `README.md` - 模块说明
- `README_CN.md` - 中文说明

### 2. 创建识别脚本
创建 `transcribe-audio.js` - 音频识别主脚本：
- 读取 output/audio/ 目录中的 WAV 文件
- 使用 VAD + Whisper 进行语音识别
- 输出结果到 output/transcription/ 目录

### 3. 功能实现
基于 `test_vad_with_non_streaming_asr_whisper_yd.js` 的逻辑：
- VAD 检测语音片段
- Whisper 实时语音识别
- 生成带时间戳的分段结果
- 生成完整文本

### 4. 更新项目计划
在 `项目计划.md` 中添加音频识别模块的说明

## 预期结果

- 创建独立的音频识别模块
- 可以批量处理 WAV 文件
- 输出分段识别结果（带时间戳）
