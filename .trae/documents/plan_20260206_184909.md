## 创建 SenseVoice 语音识别器

### 目标
创建一个新的音频识别模块，使用 SenseVoice 模型替代 Whisper，以提高识别准确度

### 实施步骤

#### 1. 创建 audio-transcriber-sense 模块
- 创建 `audio-transcriber-sense/` 目录
- 复制 `audio-transcriber/` 的文件结构
- 创建 `src/transcriber-sense.ts` 文件

#### 2. 修改 transcriber-sense.ts 支持 SenseVoice
- 替换 Whisper 为 SenseVoice 模型配置
- 修改识别器创建逻辑
- 保持 VAD + SenseVoice 的组合

#### 3. 创建 package.json 和 tsconfig.json
- 配置 TypeScript 编译
- 设置构建脚本

#### 4. 创建 README 文档
- 说明 SenseVoice 模型的使用方法
- 提供配置示例

#### 5. 创建使用脚本
- 创建 `transcribe-audio-sense.js` 脚本
- 使用 SenseVoice 模型进行批量识别
