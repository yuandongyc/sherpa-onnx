# 将视频转换功能提取到单独文件夹并使用 TypeScript

## 新增项目结构
```
English-Video-Processing/
├── video-converter/ (新文件夹)
│   ├── src/
│   │   ├── index.ts (导出接口)
│   │   └── converter.ts (视频转换逻辑)
│   ├── dist/ (编译输出)
│   ├── package.json (子包配置)
│   └── tsconfig.json (TypeScript 配置)
├── index.js (更新，使用编译后的模块)
├── package.json (更新，添加 TypeScript 依赖)
└── ...
```

## 实施步骤

### 1. 创建 video-converter 目录结构
- 创建 video-converter/src/ 目录

### 2. 创建 TypeScript 配置文件
- `video-converter/tsconfig.json`: TypeScript 编译配置
- `video-converter/package.json`: 子包配置

### 3. 创建 TypeScript 源文件
- `video-converter/src/converter.ts`: 视频转换逻辑（从 index.js 的 convertVideoToAudio 函数迁移）
- `video-converter/src/index.ts`: 导出接口

### 4. 更新主项目 package.json
- 添加 TypeScript 相关依赖（typescript, @types/node）
- 添加编译脚本

### 5. 更新 index.js
- 移除 convertVideoToAudio 函数
- 导入编译后的 video-converter 模块

### 6. 编译 TypeScript
- 运行编译命令生成 dist 目录

## TypeScript 接口定义
```typescript
export interface ConversionResult {
  wavPath: string;
  mp3Path: string;
}

export interface ConverterConfig {
  ffmpegPath: string;
  outputDir: string;
  audio: {
    sampleRate: number;
    channels: number;
    sampleFormat: string;
  };
}
```
