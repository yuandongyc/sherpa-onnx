# 视频截图参数生成功能实施计划

## 需求分析
创建一个新功能，根据 `input/` 目录中的视频文件生成截图参数的 JSON 配置文件，并新建一个文件夹来存储这些配置。

## 实施步骤

### 1. 创建截图生成模块 (screenshot-generator)
- 创建 `screenshot-generator/` 目录
- 创建 `src/` 子目录
- 创建 TypeScript 文件：
  - `screenshot-generator.ts` - 核心截图参数生成逻辑
  - `index.ts` - 导出接口
- 创建配置文件：
  - `package.json` - npm 包配置
  - `tsconfig.json` - TypeScript 配置
- 创建 README 文件：
  - `README.md` - 英文说明
  - `README_CN.md` - 中文说明

### 2. 更新 config.js
- 添加截图生成相关的配置：
  - 截图输出目录（新建 `output/screenshots/`）
  - 截图参数配置（时间间隔、截图数量等）

### 3. 创建截图参数生成脚本 (generate-screenshot-params.js)
- 扫描 `input/` 目录中的视频
- 为每个视频生成截图参数 JSON
- 输出到 `output/screenshots/` 目录

### 4. 更新 .gitignore
- 添加 `output/screenshots/` 到忽略列表

### 5. 更新项目结构文档
- 更新 `项目计划.md` 添加截图生成步骤

## 截图参数 JSON 格式示例
```json
{
  "videoPath": "input/New-Concept/video2.mp4",
  "videoName": "video2",
  "category": "New-Concept",
  "duration": 144,
  "screenshots": [
    {
      "timestamp": 10,
      "description": "Opening scene"
    },
    {
      "timestamp": 30,
      "description": "Main content"
    }
  ]
}
```
