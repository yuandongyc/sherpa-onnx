# 步骤1: 创建项目目录结构

创建以下目录结构：

```
English-Video-Processing/
├── extern/ (外部工具目录)
│   └── ffmpeg/ (FFmpeg 相关文件)
├── input/ (输入视频目录)
│   ├── New-Concept/ (新概念视频)
│   └── Travel-Across-America/ (美国之旅视频)
└── output/ (输出目录)
    ├── audio/ (转换后的音频)
    └── transcription/ (识别结果)
```

使用 PowerShell 命令创建所有必要的目录。
