# 为 video-list.json 添加 duration2 字段

## 修改内容

### 1. 更新 TypeScript 接口
在 `video-converter/src/converter.ts` 中：
- 在 `VideoFile` 接口添加 `duration2` 字段
- 类型为 `number`

### 2. 更新扫描逻辑
在 `scanDirectory` 方法中：
- 设置 `duration2` 默认值等于 `duration`
- 允许用户后续手动修改 `duration2`

### 3. 重新编译
- 运行 TypeScript 编译

### 4. 重新生成配置文件
- 运行 scan-videos.js 生成新的 video-list.json

## 预期结果

```json
{
  "videos": [
    {
      "path": "New-Concept/video2.mp4",
      "name": "video2",
      "category": "New-Concept",
      "extension": ".mp4",
      "duration": 144,
      "duration2": 144
    }
  ],
  "totalCount": 3,
  "generatedAt": "2026-02-06T12:54:07.558Z"
}
```

**说明**：
- `duration`: 视频原始时长（自动获取）
- `duration2`: 可手动修改的时长（默认等于 duration）
